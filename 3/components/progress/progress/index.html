进度条 #

基本用法 #

使用实体类progress和progress-bar应用进度条CSS组件。

40% Primary
  <div class="progress">
    <div class="progress-bar" role="progressbar" aria-valuenow="40" aria-valuemin="0" aria-valuemax="100" style="width: 40%">
      <span class="sr-only">40% Primary</span>
    </div>
  </div>
