面包屑 #

基本用法 #

使用类 .breadcrumb 来实现面包屑，通常搭配 <ol>、<li> 元素使用。

  1. 首页
  2. 图书馆
  3. 数据
<ol class="breadcrumb">
  <li><a href="your/url/">首页</a></li>
  <li><a href="your/url/">图书馆</a></li>
  <li class="active">数据</li>
</ol>
