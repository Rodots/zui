单位概念 #

root：根元素的大小。

px（pixel，像素）：相对长度单位。像素 px 数计相对于显示器屏幕分辨率而言的。

rem：CSS3 新增的一个相对单位。使用 rem 为元素设定字体大小时，相对的只是 HTML 根元素。
