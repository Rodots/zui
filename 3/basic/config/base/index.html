语义化配色表 #

名称配色语义
primaryyoung主要：主题的、可链接、正常
secondaryblue次要：次级、常态的
successgreen成功：完成、积极
warningyellow关注：提示、重点
dangerred警告：提示、异常、警醒
importantmagenta重要：优先
specialpurple特殊：触动、激情
