AjaxForm #

基于 form 组件和 Ajax 请求的表单验证插件, 用以收集、校验、提交数据。

使用 #

使用 new AjaxForm 实例创建 AjaxForm 插件。

new AjaxForm(elementId, options);

rules #

表单数据提交至接口前的校验，避免接口多次请求。格式如下：

new AjaxForm(elementId, {
  rules: {
    'key': {
      required: true, // 是否必填
      msg: 'name必填', // 错误信息提示
      patterns: [{
        reg: /^[a-zA-Z]+$/, // 前端传递的正则
        msg: 'name请填入英文', // 错误信息提示
      }],
    },
  }
  ...
});

回调 #

根据需求添加表单提交后不同状态下的所需要的逻辑。

名称参数含义作用
success调用接口成功后的返回数据表单成功的回调函数
error调用接口失败后的返回数据表单失败的回调函数
finish调用接口完成后的返回数据表单完成的回调函数
new AjaxForm(elementId, {
    rules: {
        'name': {
            required: true,
            msg: 'name必填',
            patterns: [{
                reg: /^[a-zA-Z]+$/,
                msg: 'name请填入英文',
            }],
        },
    },
    success: (data)=>{
        alert('success', data);
    },
    error: (data)=>{
        alert('fail', data);
    },
    finish: (data)=>{
        alert('finish', data);
    },
});
